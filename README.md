# 🛡️ AML Fraud Detection System

## 📌 Overview
This project is an AML (Anti Money Laundering) Fraud Detection System developed using Machine Learning, SQL, and Streamlit. It detects suspicious financial transactions using transaction patterns and rule-based fraud analysis.

---

## 🚀 Features
- Fraud Detection
- Machine Learning Prediction
- Streamlit Dashboard
- SQL AML Queries
- Real-Time User Input

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- SQL
